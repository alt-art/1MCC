# Exes and Ohs
Check to see if a string has the same amount of 'x's and 'o's. The method must return a boolean and be case insensitive. The string can contain any char.
## Examples

```javascript
Input: "ooxx"
Output: true
```

```javascript
Input: "xooxx"
Output: false
```

```javascript
Input: "ooxXm"
Output: true
```
